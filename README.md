# Code source pour SOSAlerte
Ce projet est une application web intégrée au système de gestion des urgences au Burkina Faso. SOSAlerte permet de coordonner les interventions des sapeurs-pompiers, de la police et des services de secours.

## Installation
1. Clonez le dépôt GitHub : `git clone https://github.com/AlerteSapeurWeb/alertsapeur.git`
2. Naviguez vers le répertoire du projet : `cd alertsapeur`
3. Installez les dépendances : `npm install`
4. Démarrez l'application : `npm run dev`
