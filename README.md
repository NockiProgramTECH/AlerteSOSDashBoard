# Code source pour AlerteSapeurWeb 
Ce projet est un projet de développement d'une application web pour les sapeurs-pompiers. L'application permet de gérer les interventions, les équipements, les formations, etc.
## Installation
1. Clonez le dépôt GitHub : `git clone https://github.com/AlerteSapeurWeb/alertsapeur.git`
2. Naviguez vers le répertoire du projet : `cd alertsapeur`
3. Installez les dépendances : `npm install`
4. Démarrez l'application : `npm start`
